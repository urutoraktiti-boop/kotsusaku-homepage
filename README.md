# コツコツサクサク

> **10分コツコツ。気づけばサクサク。**

## プロジェクト概要

「コツコツサクサク」は、**10分単位の学習を記録し、ゲーム感覚で習慣を続けられる学習記録 PWA**です。  
資格勉強・受験対策・自己学習など、目的を問わず「まず1マス残す」という小さな行動から始められます。

## 目標

- 「完璧にやろうとして挫折する」サイクルを断ち切り、10分という最小単位で継続をサポートする
- ストーリー・キャラクター・バッジなどの RPG 的演出で、勉強を"冒険"に変える
- 匿名の集合データで「みんなも頑張っている」ことを可視化し、孤独な学習をコミュニティ体験に変える

## 主な機能

| 機能 | 説明 |
|------|------|
| **10分ブロック記録** | タップ1つで今日の学習マスを記録。積み上がった軌跡が星になる |
| **ストーリー選択** | メインストーリー（ゴリラ・剣士・宇宙など）または資格ストーリー（簿記・宅建・FP など）を選択 |
| **レベル & バッジ** | 記録を続けることでキャラクターが成長し、バッジが解放される |
| **ライブ統計** | Firebase 経由でリアルタイムに全ユーザーの総学習時間・ストーリー分布を表示 |
| **PWA 対応** | ホーム画面に追加することでアプリ感覚で利用可能（iOS / Android / PC） |

## キャラクター

- **ニヤブロくん / ニヤブロちゃん** — 仲間を見守るニヤニヤ系キャラ
- **ウホゴリくん / ウホゴリちゃん** — 資格系のゴリゴリ本気組キャラ

## ストーリーラインナップ

**メインストーリー（フィクション）**

- 🥦 正体不明育成記 / 🐉 鬼教官の特訓 / 🦍 ゴリラ成長記 / ⚔️ 受験剣士の道 / 🚀 宇宙探検家の旅

**資格ストーリー（実際の資格対応）**

- 📗 簿記3級 / 📒 簿記2級 / 🧮 税理士試験 / ⚖️ 社労士 / 🏠 宅建 / 💰 FP2級 / 🏛️ 公務員試験 / 💼 中小企業診断士 / 🔍 公認会計士
